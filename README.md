# Aero Launcher

Un launcher entièrement optimisé pour offrir les meilleures performances de jeu. Propulsé par une architecture ultra-légère en Rust (Tauri) et conçu pour être accessible à tous, il intègre une gestion fluide des comptes Crack (Offline) et Premium (Bientôt disponible..) tout en réduisant au maximum l'impact sur les ressources de votre machine.

Vous pouvez dès maintenant télécharger et tester l'application sur Windows et macOS.

**Fonctionnalités de cette version (v0.1.0) :**

* **Performance :** Développé en Rust avec Tauri, le launcher consomme un minimum de ressources en arrière-plan.
* **Gestion de Java :** Allocation dynamique de la mémoire RAM et détection automatique de vos chemins Java.
* **Téléchargement optimisé :** Intégration native avec le moteur PortableMC pour une installation propre et rapide des versions de jeu.
* Et bien plus encore !
